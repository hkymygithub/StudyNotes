# 常用指令

### 帮助指令

#### man

```
#获得帮助信息
man [命令或配置文件]
```

#### help

```
help 命令
```

### 文件目录类

#### pwd

```
#显示当前工作路径的绝对路径
pwd
```

#### ls

```
#列出目录下内容
ls [选项] [目录或文件]
#常用选项：
#-a :显示当前目录所有文件和目录，包括隐藏的
#-l :以列表的方式显示信息
```

#### cd

```
#切换到指定目录
cd [参数]
# cd ~ 或 cd : 回到自己的家目录
# cd .. 回到当前目录的上级目录
```

#### mkdir

```
#创建目录
mkdir /test
#创建多级目录
mkdir -p /test/next
```

#### rmdir

```
#基本语法
#只能删除空目录
rmdir [选项] 要删除的目录名
rmdir /test
```

#### touch

```
#创建一个空文件
touch 文件名
```

#### cp

```
#拷贝指定文件到指定目录
cp [选项] source dest
#递归复制整个文件夹
cp -r /source /dest
#递归复制整个文件夹,并覆盖重复文件
\cp -r /source /dest
```

#### rm

```
#删除文件或目录
rm [选项] 要删除的文件或目录
#常用选项
#-r: 递归删除整个文件夹
#-f: 强制删除不提示
```

#### mv

```
#移动目录或重命名
#重命名（同一目录下）
mv oldFileName newFileName
#移动文件
mv /from /target
```

#### cat

``` 
#查看文件内容
cat [选项] 文件名
#常用选项
#-n 显示行号
```

#### more

```
#more指令是一个基于VI编辑器的文本过滤器，它会按页显示内容。more指令中内置了许多快捷键。
more 要查看的文件
```

| 操作     | 功能                 |
| -------- | -------------------- |
| 空格     | 向下翻一页           |
| Enter    | 向下翻一行           |
| q        | 离开more             |
| Ctrl + F | 向下滚动一屏         |
| Ctrl + B | 返回上一屏           |
| =        | 输出当前行号         |
| :f       | 输出文件名和当前行号 |

#### less

```
#用来分屏查看文件内容。
less 文件名
```

| 操作       | 功能                                   |
| ---------- | -------------------------------------- |
| 空格       | 向下翻一页                             |
| [pagedown] | 向下翻一页                             |
| [pageup]   | 向上翻一页                             |
| /字串      | 向下搜寻[字串]，n:向下寻找，N:向上寻找 |
| ?字串      | 向上搜寻[字串]，n:向上寻找，N:向下寻找 |
| q          | 退出                                   |

#### echo

```
#输出内容到控制台
echo [选项] [输出内容]
```

#### head

```
#显示文件开头内容，默认显示前10行。
head 文件名
#显示文件前5行
head -n 5 文件名
```

#### tail

```
#输出文件尾部内容
tail 文件名
#查看文件尾5行内容
tail -n 5 文件名
#实时追踪该文档的所有跟新
tail -f 文件名
```

#### >和>>

```
# '>' 输出重定向，'>>'追加
ls -l > 文件
ls -l >> 文件
cat 文件1 > 文件2
echo '内容' >> 文件
```

#### ln

```
#软链接
ln -s [源文件或目录] [软链接名]
#例如
ln -s /root /home/myroot
#删除软链接
rm /home/myroot
```

#### history

```
#查看已经执行过的命令
history
#查看近10条指令
history 10
#查看第10条指令
!10
```

### 时间日期类

#### date

```
#显示当前时间
date
#显示当前年份
date + %Y
#显示当前月份
date + %m
#显示当前是那一天
date + %d
#显示年月日时分秒
date "+%Y-%m-%d %H:%M:%S"
#设置日期
date -s 字符串时间
```

#### cal

```
#查看日历指令（默认显示本月）
cal
cal [选项]
#显示整年日历
cal 2020
```

### 搜索查找类

#### find

```
#将从指定目录下向下递归遍历各个子目录，将满足条件的文件或目录显示在终端。
find [搜索范围] [选项]
#在/home目录下查找hello.txt文件
find /home -name hello.txt
#查找根目录下大于200M的文件
find / -size +200M
```

| 选项            | 功能                       |
| --------------- | -------------------------- |
| -name<查询方式> | 按照指定文件名查找文件     |
| -user<用户名>   | 查找属于指定用户名所有文件 |
| -size<文件大小> | 按照指定文件大小查找文件   |

#### locate

```
#用于快速定位文件路径
locate 文件名
#locate指令基于数据库，所以第一次运行前，必须使用updatedb指令创建locate数据库。
```

#### which

```
#查看某个指令在哪个目录下
which ls
```

#### grep

``` 
#过滤查找，通常与管道"|"连用
grep [选项] 查找内容 源文件
#常用选项
#-n: 显示匹配行和行号
#-i: 忽略字母大小写
#例子:将/home/text.txt文件中hello出现的行显示
cat /home/hello.txt | grep -n "hello"
grep -n "hello" /home/hello.txt
```
